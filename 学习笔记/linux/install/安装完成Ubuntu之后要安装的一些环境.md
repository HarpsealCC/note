# **linux环境配置（持续更新）**

本文主要介绍刚刚安装完成linux（Ubuntu）系统或者子系统之后需要安装哪些环境

## **c++相关的环境与配置**

c/c++编译器
```
sudo apt-get install build-essential   // gcc/g++编译器
sudo apt install cmake  // cmake

```
java安装
```
sudo apt install openjdk-11-jdk  // 安装java
```


代码管理
```
sudo apt install subversion  // 安装svn
```
